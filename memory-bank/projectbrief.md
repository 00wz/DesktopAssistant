# DesktopAssistant - Project Brief

## Обзор проекта

DesktopAssistant — кроссплатформенный десктопный AI-агент на .NET, предоставляющий возможность вести диалоги с различными LLM провайдерами через единый интерфейс с поддержкой голосового ввода/вывода.

## Основные требования

### Функциональные требования

1. **Множественные диалоги** — поддержка нескольких диалогов в виде вкладок в UI
2. **История диалогов в виде графа** — сохранение истории с возможностью ветвления и переключения между ветками (как в ChatGPT web)
3. **Автоматическая суммаризация** — при достижении заданного размера контекста
4. **Голосовой ввод** — стриминговое распознавание речи с поддержкой wake word
5. **Голосовой вывод** — стриминговое преобразование ответов LLM в речь
6. **MCP-серверы** — поддержка подключения Model Context Protocol серверов
7. **Работа в трее** — фоновая работа с ожиданием wake word

### Настройки при первом запуске

- Выбор провайдера (OpenAI, Azure OpenAI, Anthropic и др.)
- Выбор модели
- Ввод API-ключа

### Настройки при создании диалога

- Имя ассистента (используется для wake word)
- Голос для TTS
- Системный промпт
- Дополнительные параметры модели

### UI требования

- Вкладки с именем ассистента/названием диалога
- Кнопка soft-delete на каждой вкладке
- Возможность изменения настроек провайдера

## Технический стек

### Основные технологии

| Компонент | Технология | Версия |
|-----------|------------|--------|
| UI Framework | Avalonia UI | 11.3.x |
| MVVM Toolkit | CommunityToolkit.Mvvm | 8.4.x |
| AI Orchestration | Microsoft Semantic Kernel | 1.x |
| Speech Recognition | Vosk | 0.3.x |
| Database | SQLite + EF Core | 9.x |
| Dependency Injection | Microsoft.Extensions.DI | 9.x |

### Дополнительные зависимости

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Text-to-Speech | Azure.AI.Speech / OpenAI TTS | Стриминговая генерация речи |
| Audio I/O | NAudio | Работа с аудио |
| JSON | System.Text.Json | Сериализация |
| Configuration | Microsoft.Extensions.Configuration | Конфигурация |
| Logging | Microsoft.Extensions.Logging + Serilog | Логирование |
